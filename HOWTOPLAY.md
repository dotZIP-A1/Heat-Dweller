# Heat Dweller — How to Play

## Getting Started
1. Download and extract the ZIP folder.
2. Open the extracted folder.
3. Run `heatdweller-ng.exe` to start the game.

> Note: `index.html` is included for development purposes and is not required to play the game. It must NOT be deleted.

---

## Basic Controls

- **WASD** — Move character
- **Arrow Keys** — Attack

*(Additional controls may be added in future updates.)*

---

## Saving / Progress

- Saving is not implemented yet due to the game being in early development.

---

## Troubleshooting

### The game won’t open
- Make sure you extracted the ZIP first.
- Run `heatdweller-ng.exe` directly from the folder.

### Missing files error
- Do not move the EXE out of the folder.
- Keep all assets and folders together.

### Performance issues
- Close background applications.
- Try running in windowed mode if available.
